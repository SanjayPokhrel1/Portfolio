# Dallas Hotel Forecasting Dashboard — FIFA 2026

A market-wide forecasting engine and Power BI dashboard modeling hotel performance during FIFA 2026 events in Dallas/Arlington.

## Project Summary
This project simulates real revenue‑management workflows using:
- Event-driven demand forecasting
- Compression-level classification
- ADR uplift modeling
- Hotel sensitivity factors
- Market-wide RevPAR and occupancy trends

The final dashboard is fully interactive and designed to mirror revenue management style reporting.
